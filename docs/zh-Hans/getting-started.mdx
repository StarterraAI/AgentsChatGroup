## 支持的系统

AgentsChatGroup 可在以下平台运行：

- macOS（Intel 和 Apple Silicon）
- Linux
- Windows

## 前提条件

在安装 AgentsChatGroup 之前，请确保您已安装至少一个支持的 AI 代理：

| 代理 | 安装命令 |
|-------|-------------|
| [Claude Code](https://docs.anthropic.com/en/docs/claude-code) | `npm install -g @anthropic-ai/claude-code` |
| [Gemini CLI](https://github.com/google-gemini/gemini-cli) | `npm install -g @google/gemini-cli` |
| [Codex](https://github.com/openai/codex) | `npm install -g @openai/codex` |
| [QWen Coder](https://qwenlm.github.io/qwen-code-docs/en/users/overview/) | `npm install -g @qwen-code/qwen-code@latest` |

<Tip>
您可以按照其官方文档安装其他代理。
</Tip>

## 快速入门

<Steps>
<Step title="安装编程代理">
  确保您的系统上已安装并认证至少一个[支持的编程代理](/zh-Hans/supported-coding-agents)。
</Step>

<Step title="启动 AgentsChatGroup">
  选择一种启动方式：

  **方式一：通过 npx 启动**
  ```bash
  npx agents-chatgroup
  ```

  **方式二：通过桌面安装包启动**
  从系统应用菜单中打开已安装的 **AgentsChatGroup**。

  <Check>
  应用程序绑定到一个随机空闲端口，并在终端中打印 URL。
  </Check>
</Step>

<Step title="创建聊天会话">
  点击"创建会话"开始新的群聊。这是您进行 AI 协作的工作空间。
</Step>

<Step title="添加 AI 成员">
  在右侧边栏中，点击"添加 AI 成员"将代理添加到您的会话中。为每个成员配置：
  - 唯一的 @提及名称
  - 选择代理类型（如 Claude Code、Codex）
  - 设置系统提示以定义其角色
  - 指定其工作空间路径
</Step>

<Step title="开始协作">
  使用 @提及向您的 AI 团队成员分配任务。他们将自动接收上下文并可以相互沟通！
</Step>
</Steps>

<Tip>
要使用固定端口，请指定 `PORT` 环境变量：`PORT=8080 npx agents-chatgroup`
</Tip>
