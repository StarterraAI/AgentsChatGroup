<Steps>
<Step title="运行 Qwen Code">
  ```bash
  npx -y @qwen-code/qwen-code
  ```
</Step>

<Step title="按照登录说明操作">
  按照提示完成认证流程。更多详情请参阅 [Qwen 官方文档](https://github.com/QwenLM/qwen-code)。
</Step>

<Step title="启动 agents-chatgroup">
  认证完成后，启动 agents-chatgroup：

  **方式一：通过 npx 启动**
  ```bash
  npx agents-chatgroup
  ```

  **方式二：通过桌面安装包启动**
  直接启动已安装的桌面应用。安装步骤请参阅[快速入门](/zh-Hans/getting-started)。

  现在您可以在创建任务尝试时选择 Qwen Code。
</Step>
</Steps>
