<Steps>
<Step title="安装 Cursor Agent CLI">
  ```bash
  curl https://cursor.com/install -fsS | bash
  ```

  使用 `cursor-agent --version` 验证安装。更多详情请参阅 [官方安装指南](https://docs.cursor.com/en/cli/installation)。
</Step>

<Step title="按照登录说明操作">
  使用 `cursor-agent login` 登录（打开浏览器）。您也可以设置 `CURSOR_API_KEY` 环境变量。完整说明：https://docs.cursor.com/en/cli/reference/authentication
</Step>

<Step title="启动 agents-chatgroup">
  认证完成后，启动 agents-chatgroup：

  **方式一：通过 npx 启动**
  ```bash
  npx agents-chatgroup
  ```

  **方式二：通过桌面安装包启动**
  直接启动已安装的桌面应用。安装步骤请参阅[快速入门](/zh-Hans/getting-started)。

  现在您可以在创建任务尝试时选择 Cursor Agent CLI。
</Step>
</Steps>
