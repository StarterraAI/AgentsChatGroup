<Steps>
<Step title="Exécuter GitHub Copilot">
  ```bash
  npx -y @github/copilot
  ```
</Step>

<Step title="Suivre les instructions de connexion">
  Lorsque vous y êtes invité, suivez les instructions à l'écran pour vous authentifier en utilisant la commande `/login`. Pour plus de détails, voir la [documentation GitHub Copilot CLI](https://docs.github.com/en/copilot/how-tos/use-copilot-agents/use-copilot-cli).
</Step>

<Step title="Démarrer agents-chatgroup">
  Une fois authentifié, lancez agents-chatgroup :

  **Option 1 : Démarrer avec npx**
  ```bash
  npx agents-chatgroup
  ```

  **Option 2 : Utiliser l'installateur Desktop**
  Lancez directement l'application desktop installée. Consultez le [guide de démarrage](/fr/getting-started) pour les instructions d'installation.

  Vous pouvez maintenant sélectionner GitHub Copilot lors de la création de tentatives de tâches.
</Step>
</Steps>
