## 支援的系統

AgentsChatGroup 可在以下平台執行：

- macOS（Intel 和 Apple Silicon）
- Linux
- Windows

## 前置條件

在安裝 AgentsChatGroup 之前，請確保您已安裝至少一個支援的 AI 代理：

| 代理 | 安裝指令 |
|-------|-------------|
| [Claude Code](https://docs.anthropic.com/en/docs/claude-code) | `npm install -g @anthropic-ai/claude-code` |
| [Gemini CLI](https://github.com/google-gemini/gemini-cli) | `npm install -g @google/gemini-cli` |
| [Codex](https://github.com/openai/codex) | `npm install -g @openai/codex` |
| [QWen Coder](https://qwenlm.github.io/qwen-code-docs/en/users/overview/) | `npm install -g @qwen-code/qwen-code@latest` |

<Tip>
您可以按照其官方文檔安裝其他代理。
</Tip>

    **方式一：使用 npx**

    npx agents-chatgroup

    **方式一：使用 npx**

    ```bash
    npx agents-chatgroup
    ```

    **方式二：桌面安裝包（推薦）**

    從 Releases 下載 Linux 安裝包：
    <a href="https://github.com/StarterraAI/AgentsChatGroup/releases/latest" target="_blank" rel="noopener noreferrer">

## 快速入門

<Steps>
<Step title="安裝程式設計代理">
  確保您的系統上已安裝並認證至少一個[支援的程式設計代理](/zh-Hant/supported-coding-agents)。
</Step>

<Step title="啟動 AgentsChatGroup">
  選擇一種啟動方式：

  **方式二：透過 npx 啟動**
  ```bash
  npx agents-chatgroup
  ```

  **方式一：透過桌面安裝包啟動**
  從系統應用程式選單開啟已安裝的 **AgentsChatGroup**。

  <Check>
  應用程式綁定到一個隨機空閒連接埠，並在終端機中列印 URL。
  </Check>
</Step>

<Step title="建立聊天會話">
  點擊「建立會話」開始新的群聊。這是您進行 AI 協作的工作空間。
</Step>

<Step title="新增 AI 成員">
  在右側邊欄中，點擊「新增 AI 成員」將代理新增到您的會話中。為每個成員配置：
  - 唯一的 @提及名稱
  - 選擇代理類型（如 Claude Code、Codex）
  - 設定系統提示以定義其角色
  - 指定其工作空間路徑
</Step>

<Step title="開始協作">
  使用 @提及向您的 AI 團隊成員分配任務。他們將自動接收上下文並可以相互溝通！
</Step>
</Steps>

<Tip>
要使用固定連接埠，請指定 `PORT` 環境變數：`PORT=8080 npx agents-chatgroup`
</Tip>
