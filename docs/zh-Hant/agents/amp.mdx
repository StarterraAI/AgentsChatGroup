<Steps>
<Step title="執行 Amp">
  ```bash
  npx -y @sourcegraph/amp
  ```
</Step>

<Step title="按照登入說明操作">
  按照提示完成認證流程。更多詳情請參閱 [Amp 使用者手冊](https://ampcode.com/manual#install)。
</Step>

<Step title="啟動 agents-chatgroup">
  認證完成後，啟動 agents-chatgroup：

  **方式一：透過 npx 啟動**
  ```bash
  npx agents-chatgroup
  ```

  **方式二：透過桌面安裝包啟動**
  直接啟動已安裝的桌面應用。安裝步驟請參閱[快速入門](/zh-Hant/getting-started)。


  現在您可以在建立任務嘗試時選擇 Amp。
</Step>
</Steps>
