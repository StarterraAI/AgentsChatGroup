<Steps>
<Step title="Run Amp">
  ```bash
  npx -y @sourcegraph/amp
  ```
</Step>

<Step title="Follow the login instructions">
  Complete the authentication flow as prompted. For more details, see the [Amp Owner's Manual](https://ampcode.com/manual#install).
</Step>

<Step title="Start agents-chatgroup">
  Once authenticated, launch agents-chatgroup:

  **Option 1: Run with npx**
  ```bash
  npx agents-chatgroup
  ```

  **Option 2: Use Desktop Installer**
  Launch the installed desktop app directly. See [Getting Started](/getting-started) for installer instructions.


  You can now select Amp when creating task attempts.
</Step>
</Steps>
