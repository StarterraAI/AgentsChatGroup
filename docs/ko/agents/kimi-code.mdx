<Steps>
<Step title="Kimi Code 실행">
  Mac OS 및 Linux:
  ```bash
  # Linux / macOS
  curl -LsSf https://code.kimi.com/install.sh | bash
  kimi
  ```
  Windows:
  ```powershell
  # Windows (PowerShell)
  Invoke-RestMethod https://code.kimi.com/install.ps1 | Invoke-Expression
  kimi
  ```
</Step>

<Step title="로그인 지침 따르기">
  프롬프트에 따라 인증 흐름을 완료합니다. 자세한 내용은 [Kimi 공식 문서](https://moonshotai.github.io/kimi-cli/en/guides/getting-started.html)를 참조하세요.
</Step>

<Step title="agents-chatgroup 시작">
  인증이 완료되면 agents-chatgroup을 시작합니다:

  **방법 1: npx로 시작**
  ```bash
  npx agents-chatgroup
  ```

  **방법 2: Desktop Installer로 시작**
  설치된 데스크톱 앱을 직접 실행할 수 있습니다. 설치 방법은 [시작하기](/ko/getting-started)를 참고하세요.

  이제 작업 시도를 생성할 때 Kimi Code를 선택할 수 있습니다.
</Step>
</Steps>
