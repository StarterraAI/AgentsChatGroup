<Steps>
<Step title="OpenCode 실행">
  ```bash
  npx -y opencode-ai
  ```
</Step>

<Step title="로그인 지침 따르기">
  프롬프트에 따라 인증 흐름을 완료합니다. 자세한 내용은 [OpenCode GitHub 페이지](https://github.com/sst/opencode)를 참조하세요.
</Step>

<Step title="agents-chatgroup 시작">
  인증 완료 후 agents-chatgroup 시작:

  **방법 1: npx로 시작**
  ```bash
  npx agents-chatgroup
  ```

  **방법 2: Desktop Installer로 시작**
  설치된 데스크톱 앱을 직접 실행할 수 있습니다. 설치 방법은 [시작하기](/ko/getting-started)를 참고하세요.

  이제 작업 시도를 생성할 때 OpenCode를 선택할 수 있습니다.
</Step>
</Steps>
