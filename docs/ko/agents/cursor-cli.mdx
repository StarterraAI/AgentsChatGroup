<Steps>
<Step title="Cursor Agent CLI 설치">
  ```bash
  curl https://cursor.com/install -fsS | bash
  ```

  `cursor-agent --version`으로 설치 확인. 자세한 내용은 [공식 설치 가이드](https://docs.cursor.com/en/cli/installation)를 참조하세요.
</Step>

<Step title="로그인 지침 따르기">
  `cursor-agent login`으로 로그인 (브라우저 열림). `CURSOR_API_KEY` 환경 변수를 설정할 수도 있습니다. 전체 지침: https://docs.cursor.com/en/cli/reference/authentication
</Step>

<Step title="agents-chatgroup 시작">
  인증 완료 후 agents-chatgroup 시작:

  **방법 1: npx로 시작**
  ```bash
  npx agents-chatgroup
  ```

  **방법 2: Desktop Installer로 시작**
  설치된 데스크톱 앱을 직접 실행할 수 있습니다. 설치 방법은 [시작하기](/ko/getting-started)를 참고하세요.

  이제 작업 시도를 생성할 때 Cursor Agent CLI를 선택할 수 있습니다.
</Step>
</Steps>
