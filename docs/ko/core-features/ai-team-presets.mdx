AI 팀 프리셋을 사용하면 AI 멤버 구성을 저장하고 재사용할 수 있어, 매번 처음부터 설정하지 않아도 됩니다.

프리셋은 두 가지입니다.

- **멤버 프리셋** - 단일 AI 멤버 설정(역할, 역할 설정, 에이전트 유형)
- **팀 프리셋** - 여러 멤버 프리셋 조합을 세션에 한 번에 가져오기

## 내장 프리셋

AgentsChatGroup에는 현재 **멤버 프리셋 22개**와 **팀 프리셋 8개**가 내장되어 있습니다.

## 프리셋 관리

**설정 -> 프리셋**에서 멤버/팀 템플릿을 관리할 수 있습니다.

### 멤버 프리셋 탭

- **추가** - 새 커스텀 멤버 프리셋 생성
- **수정** - 기존 커스텀 프리셋 수정
- **복사** - 내장/커스텀 프리셋을 기반으로 새 프리셋 생성
- **비활성화** - 삭제 없이 빠른 추가 목록에서 숨김
- **삭제** - 더 이상 필요 없는 커스텀 프리셋 삭제

### 팀 프리셋 탭

- **추가** - 멤버를 선택해 새 팀 생성
- **수정** - 커스텀 팀의 멤버 구성 변경
- **복사** - 팀 프리셋 변형 생성
- **비활성화** - 가져오기 목록에서 숨김
- **삭제** - 커스텀 팀 삭제

## 세션에서 프리셋 사용

### 멤버 프리셋 추가

채팅 오른쪽 **AI Members 사이드바**에서:

1. 멤버 목록 위의 **멤버 빠른 추가** 클릭
2. 멤버 프리셋 선택
3. 클릭 즉시 현재 세션에 추가

추가 후에도 작업 경로, 실행기 등 설정을 편집할 수 있습니다.

### 팀 프리셋 가져오기

1. AI Members 사이드바에서 **Import Team** 클릭
2. 가져올 팀 프리셋 선택
3. **가져오기 미리보기** 확인:
   - **Created** - 새 멤버 생성
   - **Reused** - 동일 이름/동일 설정 멤버 재사용
   - **Auto-renamed** - 이름 충돌 + 설정 다름, `_2` 등 접미사 자동 부여
4. 가져오기 확정

<Tip>
미리보기 단계에서 이름 충돌을 사전에 확인할 수 있습니다. 설정이 다르면 자동으로 새 이름이 부여됩니다.
</Tip>

### 충돌 처리 규칙

| 상황 | 처리 |
|---|---|
| 동일 이름 + 동일 설정 | 기존 멤버 재사용 |
| 동일 이름 + 다른 설정 | `_2`(또는 `_3` 등) 붙여 새 멤버 생성 |
| 멤버가 이미 세션에 있음 | 기존 멤버 유지 |

## 보관된 세션

보관된 세션에서는 **가져오기 기능이 비활성화** 됩니다. 멤버 추가/팀 가져오기를 할 수 없습니다.
