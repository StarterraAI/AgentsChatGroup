<Steps>
<Step title="Qwen Codeを実行">
  ```bash
  npx -y @qwen-code/qwen-code
  ```
</Step>

<Step title="ログイン手順に従う">
  プロンプトに従って認証フローを完了します。詳細は[Qwen公式ドキュメント](https://github.com/QwenLM/qwen-code)を参照してください。
</Step>

<Step title="agents-chatgroupを起動">
  認証完了後、agents-chatgroupを起動：

  **方法1：npxで起動**
  ```bash
  npx agents-chatgroup
  ```

  **方法2：Desktop Installerで起動**
  インストール済みのデスクトップアプリを直接起動できます。インストーラー手順は[はじめに](/ja/getting-started)を参照してください。

  タスク試行を作成する際にQwen Codeを選択できるようになりました。
</Step>
</Steps>
