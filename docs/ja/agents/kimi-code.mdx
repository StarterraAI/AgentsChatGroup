<Steps>
<Step title="Kimi Codeを実行">
  Mac OS と Linux:
  ```bash
  # Linux / macOS
  curl -LsSf https://code.kimi.com/install.sh | bash
  kimi
  ```
  Windows:
  ```powershell
  # Windows (PowerShell)
  Invoke-RestMethod https://code.kimi.com/install.ps1 | Invoke-Expression
  kimi
  ```
</Step>

<Step title="ログイン手順に従う">
  プロンプトに従って認証フローを完了します。詳細については、[Kimi公式ドキュメント](https://moonshotai.github.io/kimi-cli/en/guides/getting-started.html)を参照してください。
</Step>

<Step title="agents-chatgroupを起動">
  認証が完了したら、agents-chatgroupを起動します：

  **方法1：npxで起動**
  ```bash
  npx agents-chatgroup
  ```

  **方法2：Desktop Installerで起動**
  インストール済みのデスクトップアプリを直接起動できます。インストーラー手順は[はじめに](/ja/getting-started)を参照してください。

  タスク試行を作成する際にKimi Codeを選択できるようになりました。
</Step>
</Steps>
