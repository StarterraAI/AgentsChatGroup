title: "Cursor Agent CLI"
description: "Cursorのコマンドラインエージェントをセットアップ"

---

<Steps>
<Step title="Cursor Agent CLIをインストール">
  ```bash
  curl https://cursor.com/install -fsS | bash
  ```

  `cursor-agent --version`でインストールを確認。詳細は[公式インストールガイド](https://docs.cursor.com/en/cli/installation)を参照してください。
</Step>

<Step title="ログイン手順に従う">
  `cursor-agent login`でサインイン（ブラウザが開きます）。`CURSOR_API_KEY`環境変数を設定することもできます。完全な手順：https://docs.cursor.com/en/cli/reference/authentication
</Step>

<Step title="agents-chatgroupを起動">
  認証完了後、agents-chatgroupを起動：

  **方法1：npxで起動**
  ```bash
  npx agents-chatgroup
  ```

  **方法2：Desktop Installerで起動**
  インストール済みのデスクトップアプリを直接起動できます。インストーラー手順は[はじめに](/ja/getting-started)を参照してください。

  タスク試行を作成する際にCursor Agent CLIを選択できるようになりました。
</Step>
</Steps>
