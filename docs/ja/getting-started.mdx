## サポートシステム

AgentsChatGroupは以下のプラットフォームで実行できます：

- macOS（Intel および Apple Silicon）
- Linux
- Windows

## 前提条件

AgentsChatGroupをインストールする前に、少なくとも1つのサポートされているAIエージェントがインストールされていることを確認してください：

| エージェント | インストール |
|-------|-------------|
| [Claude Code](https://docs.anthropic.com/en/docs/claude-code) | `npm install -g @anthropic-ai/claude-code` |
| [Gemini CLI](https://github.com/google-gemini/gemini-cli) | `npm install -g @google/gemini-cli` |
| [Codex](https://github.com/openai/codex) | `npm install -g @openai/codex` |
| [QWen Coder](https://qwenlm.github.io/qwen-code-docs/en/users/overview/) | `npm install -g @qwen-code/qwen-code@latest` |

<Tip>
公式ドキュメントに従って他のエージェントをインストールできます。
</Tip>

    **方法1：npxを使用**


    **方法2：Desktop Installer（推奨）**

    最新のインストーラー（`.msi`）をダウンロード：

  <Tab title="macOS">
    **方法1：npxを使用**

    ```bash
    npx agents-chatgroup
    ```

    **方法2：Desktop Installer（推奨）**

    <a href="https://github.com/StarterraAI/AgentsChatGroup/releases/latest" target="_blank" rel="noopener noreferrer">

    **方法1：npxを使用**

    ```bash

    ```

    **方法2：Desktop Installer（推奨）**

## クイックスタート

<Steps>
<Step title="コーディングエージェントをインストール">
  システムに少なくとも1つの[サポートされているコーディングエージェント](/ja/supported-coding-agents)がインストールされ、認証されていることを確認してください。
</Step>

<Step title="AgentsChatGroupを起動">
  起動方法を選択してください：

  **方法2：npxで起動**
  ```bash
  npx agents-chatgroup
  ```

  **方法1：Desktop Installerで起動**
  システムのランチャーから、インストール済みの **AgentsChatGroup** を開きます。

  <Check>
  アプリケーションはランダムな空きポートにバインドし、ターミナルにURLを表示します。
  </Check>
</Step>

<Step title="チャットセッションを作成">
  「セッション作成」をクリックして新しいグループチャットを開始します。これがAIコラボレーションのワークスペースです。
</Step>

<Step title="AIメンバーを追加">
  右サイドバーで「AIメンバーを追加」をクリックしてエージェントをセッションに追加します。各メンバーに以下を設定：
  - ユニークな@メンション名
  - エージェントタイプを選択（例：Claude Code、Codex）
  - 役割を定義するシステムプロンプトを設定
  - ワークスペースパスを指定
</Step>

<Step title="コラボレーション開始">
  @メンションを使用してAIチームメンバーにタスクを割り当てます。彼らは自動的にコンテキストを受け取り、互いにコミュニケーションできます！
</Step>
</Steps>

<Tip>
固定ポートを使用するには、`PORT`環境変数を指定：`PORT=8080 npx agents-chatgroup`
</Tip>
